So funktionierts

In 4 Schritten zum smarten Filament-Management

Einfach im Browser öffnen und loslegen – keine Installation nötig.

1

Öffnen & loslegen

Öffne Spool Tag Propus im Browser und installiere es optional als App (PWA) – kein App Store nötig. Wer selbst hosten möchte: Docker Compose in wenigen Minuten.

2

Einloggen & verbinden

Erstelle deinen Account, gib optional deine Spoolman-URL ein – und alle deine Spulen werden automatisch synchronisiert.

3

Filament wählen & Tag erstellen

Suche in der Datenbank nach deinem Filament, passe Farbe und Temperaturen an und schreibe die Daten auf einen NFC-Tag oder QR-Code.

4

Scannen & drucken

Smartphone ans Tag halten – dein Drucker erkennt das Filament automatisch. Slicer-Profile direkt herunterladen und loslegen.

FAQ

Häufige Fragen

Ist Spool Tag Propus kostenlos? +
Ja, Spool Tag Propus ist vollständig kostenlos und Open Source. Du kannst es direkt im Browser nutzen – oder optional selbst auf deinem Server, NAS oder Raspberry Pi hosten.
Welche NFC-Tags werden unterstützt? +
Es werden standard NDEF-kompatible NFC-Tags benötigt (NTAG213/215/216). Diese gibt es günstig bei Amazon oder AliExpress. NFC schreiben und lesen funktioniert nur auf Android-Geräten mit Chrome-Browser, da iOS die NFC Web API nicht unterstützt.
Wie installiere ich Spool Tag Propus? +
Einfach die URL im Browser öffnen – fertig. Wer selbst hosten möchte: Installation via Docker Compose. Ein fertiges docker-compose.yml findest du im Repository. Mit einem einzigen Befehl sind Frontend (Nginx) und Backend (Python/Flask) einsatzbereit – auch auf Synology NAS, UGREEN NAS oder einem Raspberry Pi.
Funktioniert die App ohne Internetzugang? +
Ja! Als Progressive Web App (PWA) mit Service Worker können alle Kernfunktionen – inkl. der Filament-Datenbank – offline genutzt werden. Lediglich die Datenbankaktualisierung und Spoolman-Sync benötigen eine Netzwerkverbindung.
Kann ich Spool Tag Propus auf dem Handy nutzen? +
Ja! Die App ist vollständig mobil-optimiert und kann als PWA auf Android und iOS installiert werden. NFC-Tag schreiben/lesen ist auf Android mit Chrome verfügbar. QR-Code-Scanning funktioniert auf allen Geräten mit Kamera.
Wie werden meine Daten gespeichert? +
Alle Daten werden lokal in einer SQLite-Datenbank auf deinem eigenen Server gespeichert. Es werden keine Daten an externe Dienste übertragen (außer du aktivierst GitHub-Issue-Erstellung für Fehlerberichte).
Gibt es einen Unterschied zwischen OpenSpool und OpenPrintTag? +
Beide sind offene NFC-Standards für Filament-Tags. OpenSpool ist einfacher und fokussiert auf grundlegende Materialdaten (Hersteller, Material, Farbe, Temperaturen). OpenPrintTag ist ein erweitertes Format mit zusätzlichen Parametern wie Flussverhältnis, maximaler volumetrischer Geschwindigkeit und mehr – ideal für fortgeschrittene Setups.
