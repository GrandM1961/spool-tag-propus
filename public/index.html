ℹ️ Über Spool Propus

⚠️ BETA — Dieses Projekt befindet sich in aktiver Entwicklung

Spool Propus ist ein Open-Source Filament Tag Manager für 3D-Drucker. Erstelle, lese und verwalte NFC-Tags und QR-Codes für deine Filament-Spulen — kompatibel mit OpenSpool und OpenPrintTag.

Features

📝
NFC Tags Lesen & Schreiben Beschreibe NFC-Tags direkt vom Handy (Android Chrome) mit Filament-Daten in OpenSpool (JSON) oder OpenPrintTag (CBOR) Format. Kompatibel mit Bambu Lab, OrcaSlicer und allen OpenSpool-fähigen Druckern.
📱
QR-Code Generator & Scanner Generiere QR-Codes mit deinen Filament-Daten zum Ausdrucken und Aufkleben auf die Spule. Scanne QR-Codes mit der Kamera, um Daten sofort zu laden — funktioniert auf allen Geräten.
🔍
Filament Datenbank Durchsuche über 13.000 Farbvarianten von 126+ Marken aus der Open Filament Database. Wähle ein Filament und erstelle direkt einen Tag.
⬇️
Slicer Profile 4.000+ Filament-Profile für OrcaSlicer und Bambu Studio — automatisch von GitHub synchronisiert. Lade Profile direkt herunter, gefiltert nach Hersteller, Material und Drucker.
📦
Spoolman Integration Verbinde dich mit deiner Spoolman-Instanz, durchsuche dein Spulen-Inventar und importiere Filament-Daten direkt in einen Tag.
🌡️
Trocknungsprofile Empfohlene Trocknungstemperaturen und -zeiten für 20+ Materialien — von PLA bis PEEK. Mit Feuchtigkeitsempfindlichkeit und Hinweisen.
🎨
Farbpalette & Marken-Filter 45 vordefinierte Filament-Farben. Materialien und Varianten werden automatisch auf die gewählte Marke gefiltert (z.B. Bambu Lab zeigt nur deren Materialien).
📲
Progressive Web App (PWA) Installierbar auf dem Handy-Homescreen. Funktioniert offline für Tag-Erstellung und Trocknungsprofile.
☀️
Hell- & Dunkelmodus Umschaltbar über den Button oben rechts. Deine Einstellung wird gespeichert.

Plattform-Unterstützung

NFC Schreiben: Android Chrome (HTTPS erforderlich)

QR-Code Scanner: Chrome 83+, Safari 17.2+, Edge 83+

Datei-Download: Alle Browser — Datei mit NFC Tools App auf Tag schreiben (iOS & Android)

Web Share: Mobile Browser mit Web Share API
